Detailed Description

std::string-like Heap-allocated RAII wrapper for bench::String.

This is conceptually very similar to the standard std::string - it stores a heap-allocated bench::String that is freed once the string goes out of scope.

bench::HeapString can be easily constructed from a bench::String and implicitly casts down to bench::String so you can easily pass it to all functions that expect a bench::String.
